### How to Enable Unknown Sources on Android Auto

To use apps that are not from the Google Play Store on Android Auto, you must unlock the hidden **Developer Settings**. Follow these concise steps:

#### 1. Open Android Auto Settings
* Go to **Settings** on your Android phone.
* Search for **"Android Auto"** and select it.

#### 2. Unlock Developer Mode
* Scroll to the very bottom to find the **Version** and permission info.
* **Tap the "Version" section 10 times** repeatedly.
* When the pop-up appears asking to allow developer settings, tap **OK**.

#### 3. Open Developer Settings
* Tap the **three-dot menu (‚ãÆ)** in the top-right corner of the screen.
* Select **Developer settings**.

#### 4. Enable Unknown Sources
* Scroll down to find the **Unknown sources** option.
* **Check the box** to enable it.
